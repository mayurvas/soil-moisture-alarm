Soil Moisture Based Plant Watering Alarm

Objective:-
To alert when soil becomes dry using Arduino and a moisture sensor.

Components Used:-

Arduino Uno R3

Soil moisture sensor(probe and module)

Buzzer(Active buzzer)

Jumper wires

Working Principle
The sensor probe reads soil moisture.
If soil is dry → buzzer making beep + second LED gets ON(two LED gets ON at a time).
If soil is wet → buzzer stops beep  + second LED gets OFF(One LED gets ON at a time).

Applications

Home gardening

Plant monitoring

Smart agriculture basics
